HTML-Overview

This html file, serves to give you a first overview about html. It gives you some tools that might be helpful for programming the survey. You may just compare the code with the output that you can see when starting your otree project in the localhost environment. If you are interested in further material that is not covered here, you might find this website helpful: HTML-Tutorial.

Headings

To add a heading one uses the <hx> </hx> tag. For the x you just insert a number between 1 and 6, the larger the number, the smaller the heading will be.

This is another heading

While the first heading is created with <h3> </h3>, the second one is done with <h5> </h5>. You can see directly see that the second one is way smaller than the first one.

Paragraphs

<p> text </p>

To create a paragraph use the tag <p> and close it with the tag </p> . Everything that is written between these two tags will appear in your paragraph. When using the paragraph text you will have no blank lines between your text . To see an example code just check out the Html_overview file in your otree project!

If you put create another paragraph it will appear just below the first one, note that there is a space between two paragraphs.

Images

To include an image in the html-file, it must be stored in the right place of your otree project. The right folder is the "images"-folder, which is the subfolder of the static folder. As a little reminder, the picture below shows the folder structure of an otree project.


A possibility to include an image is to use the following code snippet:

<img width="50%" height="50%" src=" { % static 'images/pictureName' % }" />

Of course you can vary the sizes of width and height to increase or decrease the picture size.

See Images in otree for further information

Remarks

Make sure that every part of your code is written within a block (this also includes comments!), otherwise your code will not run and you will receive the error message: "TemplateSyntaxError: All content must be inside a block tag".

A block has the following structure:

to open a block: { % block WhatIsBlockUsedFor % }

to close the block { % endblock % }

If you are using a style block (see the first block in this html file), you need to include the style tag in which you embed your code.



{% endblock %}
